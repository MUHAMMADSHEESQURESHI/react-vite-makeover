# HelpHub AI - Community Support Network 🚀

HelpHub AI is a full-stack MERN application built to bridge the gap between people seeking help and community members willing to provide it. This project was developed as part of the **SMIT (Saylani Mass IT Training) Grand Coding Night 2026**.

## 🌐 Live Demo

**Backend API:** [https://my-backend-1u2159anu-muhammad-sheess-projects.vercel.app/api](https://my-backend-1u2159anu-muhammad-sheess-projects.vercel.app/api)

---

## 🛠 Tech Stack

- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Deployment:** Vercel (Frontend & Backend)
- **Developer Tools:** Claude Code (via Ollama)

---

## ✨ Features

- **Create Help Requests:** Users can post requests with a title, description, category, and urgency level.
- **AI Suggestions:** Integrated AI logic to help refine and strengthen help descriptions.
- **Real-time Database:** All requests are stored securely in a MongoDB cloud database.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using Tailwind CSS.
- **CORS Enabled:** Secure communication between frontend and backend hosted on different domains.
